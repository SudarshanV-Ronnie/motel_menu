# Motel Menu System (C)

This is a menu-driven motel food ordering system written in C.

## Features
- Displays food menu
- Allows multiple orders
- Calculates total bill
- Simple user-friendly interface

## Language Used
- C Programming

## How to Run
1. Compile using GCC:
   gcc motel_menu.c -o motel
2. Run:
   ./motel
